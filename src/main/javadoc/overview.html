This package contains a Java 1.7 Parser with AST (Abstract Syntax Tree) generation and visitor support. The AST records the source code structure, javadoc and comments.

This parser is based on Sreenivasa Viswanadha's Java 1.5 parser. The project was originally hosted at http://code.google.com/p/javaparser/ but seemed dead. This repository at https://github.com/rpau/javalang keeps the code alive for walkmod because it is modified according the walkmod needs.

Javadoc texts can be also parseable to process Javadoc tags.

The Java AST nodes can be merged to simplify code transformations, using a semantic merge engine.
